# Vue Map Maker

This is a simple creator of maps. Feel free to use it or upgrade.

## Getting started
To install dependencies just run command: ``npm install``

To run dev server: ``npm run dev``

## Used libraries
- Vue.js
- Leaflet
- Tailwindcss
- DaisyUi
- Hero Icons
- Vite
